# docs/

Place a photo of your device here named `preview.jpg` to display it in the README.

**Recommended:** A photo of the CYD display showing the power flow dashboard in action.

To add your photo:
1. Take a photo of the running device
2. Save it as `preview.jpg` in this folder
3. Commit and push:
   ```bash
   git add docs/preview.jpg
   git commit -m "Add device preview photo"
   git push
   ```
